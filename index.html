How to get your token

  1. Open Discord in your web browser and login
  2. Press Ctrl+Shift+I (⌥⌘I on macOS) to open developer tools
  3. Press Ctrl+Shift+M (⇧⌘M) to toggle device toolbar
  4. Navigate to the Application tab
  5. On the left, expand Local Storage and select https꞉//discord.com
  6. Type token into the Filter box
  7. If the token key does not appear, press Ctrl+R (⌘R) to reload
  8. Copy the value of the token after the first dot for example D9T3mbyprC9vqnBu8PbW83nG.GEbMpd.bNBc-StwWRDKP5eY0iNkfm_a2SS copy from GebMpd to the end

How to get an user ID?

  1. Open your discord settings and go to "Appearance"
  2. Check the "Developer Mode" checkbox
  3. Save your settings
  4. Now right click the user you want to resolve and click "Copy ID"

What Is A Token?

  1. A token is a part of discord in which stores your login data and more. To break it down:
  2. First Half Ex: | D9T3mbyprC9vqnBu8PbW83nG | -- The Part Which Contains Your Login Data (VALUABLE, DONT SHARE)
  3. Second Half Part 1 Ex: | GEbMpd | -- The part which contains your bytecode (when you send a message it shows your bytecode not valuable)
  4. Second Half Part 2 Ex: | bNBc-StwWRDKP5eY0iNkfm_a2SS | -- The part which sends and recieves packets (most important for this program but otherwise useless, it will be able to send false packets to get the targets IP)

What is a Discord IP resolver and how does it work?

A Discord resolver is a tool which uses an advanced state-of-the-art technology performing a packet interception scan method which scans to extract, decrypt and fetch IP addresses of users. It sends spoofed packets modified through discords algorithms to adapt to the geo location of the target user, forcing the target to send an encrypted packet back which is used to extract information.


Why does it need the second half of your token?

You cannot login with half of a token. The system needs an authentic discord account databyte to be able to send the packets to a user however tokens change and so it would not be reliable to use one directly from our servers
